Auth configuration 比较多，未来也会扩展更多，不要在侧面栏下面处理，在右边主展示区展示和操作.

## Github Authentic Configuration

包含配置：
* Client ID
* Client Secret
* Homepage URL
* Authorization callback URL

设置这些字段，而且要告诉用户对应的环境变量名称是什么

* NextAuth secret 自动为用户生成
